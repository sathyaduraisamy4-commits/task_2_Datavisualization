TASK 2 - Data Visualization & Storytelling 
This repository contains the solution for task 2, which focuses on creating meaningful visualizations using the sales dataset.
Visualizations Created:
1.Monthly Sales Trend - shows how sales change month by month
2.Sales by Region - Compares sales across all regions.
3.Sales by sub-Category - Identifies which product groups have the highest sales.

Insights:
Monthly sales fluctuate and show seasonal behavior.
One region performs significantly better than the others.
Only a few product sub-Categories contribute majority of totalsales.
